    Introduction

    Welcome to the Project CTF² Server API Documentation!

    This document describes the usage of the endpoints exposed by the Project CTF² Server, and may be of interest to you if you

    • Are creating a custom client / frontend / interface for Project CTF²
    • Are contributing to Project CTF²'s development
    • Are bored

    auth

    login

    POST /auth/login

    Authenticates a user

    Parameter Type Description
    username string Username
    password string Password

    register

    POST /auth/register

    Registers a new user

    Parameter Type Description
    name string Display name
    username string Username
    password string Password

    me

    GET /auth/me

    Get user details

    Requirements

    • User must be logged in

    PUT /auth/me

    Update user details

    Requirements

    • User must be logged in
    Parameter Type Description
    name string New display name
    password string New password

    usernameAvailable

    POST /auth/usernameAvailable

    Check if a username is available

    Parameter Type Description
    username string Username to check

    nameAvailable

    POST /auth/nameAvailable

    Check if a name is available

    Parameter Type Description
    name string Name to check

    delete

    POST /auth/delete

    Delete a user

    Requirements

    • User must be logged in
    • User must be an admin
    Parameter Type Description
    username string Username

    all

    GET /auth/all

    Gets all user data

    Requirements

    • User must be logged in
    • User must be an admin

    ctf

    category

    GET /ctf/category

    Get categories

    Requirements

    • User must be logged in

    POST /ctf/category

    Create a category

    Requirements

    • User must be logged in
    • User must be an admin
    Parameter Type Description
    name string Category

    DELETE /ctf/category

    Delete a category

    Requirements

    • User must be logged in
    • User must be an admin
    Parameter Type Description
    id integer Category ID

    PUT /ctf/category

    Rename a category

    Requirements

    • User must be logged in
    • User must be an admin
    Parameter Type Description
    id integer Category ID
    name string Category Name

    question

    GET /ctf/question

    Get questions

    Requirements

    • User must be logged in

    POST /ctf/question

    Create a question

    Requirements

    • User must be logged in
    • User must be an admin
    Parameter Type Description
    title string Title
    description string Description
    category integer Category ID
    flag string Flag
    value integer Value

    PUT /ctf/question

    Update a question

    Requirements

    • User must be logged in
    • User must be an admin
    Parameter Type Description
    id integer Question ID
    title string Title
    description string Description
    category integer Category ID
    flag string Flag
    value integer Value

    DELETE /ctf/question

    Delete a question

    Requirements

    • User must be logged in
    • User must be an admin
    Parameter Type Description
    id integer Question ID

    getFlag

    POST /ctf/question/getFlag

    Get a question flag

    Requirements

    • User must be logged in
    • User must be an admin
    Parameter Type Description
    id integer Question ID

    solve

    POST /ctf/question/solve

    Solve a question

    Requirements

    • User must be logged in
    Parameter Type Description
    id integer Question ID
    flag string Flag

    DELETE /ctf/question/solve

    Unsolve a question

    Requirements

    • User must be logged in
    • User must be an admin
    Parameter Type Description
    questionId integer Question ID
    userId integer User ID

    stats

    scores

    GET /stats/scores

    Get all player scores

    top

    GET /stats/scores/top

    Get the top 10 players/teams

    GET /stats/scores/top/:n

    Get the top n players/teams

    Parameter Type Description
    n integer Number of players/teams

    solves

    GET /stats/solves

    Get all question solves

    GET /stats/solves/:id

    Get players who have solved a specific question

    Parameter Type Description
    id integer Question ID

    hello

    GET /hello

    Server health check
